## Running the Web Application

1. Navigate to the web application directory:

   ```sh
   cd webApp
   ```

2. Install the dependencies:

   ```sh
   pnpm i
   ```

3. Start the development server:
   ```sh
   pnpm run dev
   ```

## Changing API Base Routes

1. Open the `routes.ts` file.

2. Comment or uncomment the relevant environment file.

   - The default enabled file is `development.environment.ts`.
